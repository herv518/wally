# Memory (Langzeit, kurz und nuetzlich)

## Zweck
Memory speichert nur entscheidungsrelevantes Wissen fuer Kontinuitaet.

## Struktur
- project_vision: 1-3 Saetze zum Nordstern
- decisions: wichtige Entscheidungen mit Datum + Grund
- open_questions: offene Punkte, die Fortschritt blockieren
- next_actions: naechste 3 konkreten Schritte
- risk_register: Risiken + Gegenmassnahmen
- glossary: feste Begriffe (z. B. Key-to-Drive, Fahrzeugakte)

## Regeln
- knapp schreiben, kein Roman
- nur Fakten und verifizierte Annahmen
- keine Secrets speichern
- bei Widerspruechen: alten Eintrag markieren, nicht still ersetzen
